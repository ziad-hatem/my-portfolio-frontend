## ðŸ”§ Troubleshooting

### "Server configuration error"
**Problem:** `ADMIN_CONGRATS_PASSWORD` not set

**Solution:**
```bash
# Check .env.local
echo $ADMIN_CONGRATS_PASSWORD

# Or add it:
echo "ADMIN_CONGRATS_PASSWORD=yourpassword" >> .env.local
```

### "Failed to connect to MongoDB"
**Problem:** MongoDB connection issue

**Solution:**
- Check `MONGODB_URI` in `.env`
- Verify MongoDB Atlas IP whitelist
- Test connection: `mongosh "YOUR_MONGODB_URI"`

### Image upload not working
**Problem:** Image too large or wrong format

**Solution:**
- Max size: 5MB
- Formats: JPEG, PNG, WebP, GIF
- Try compressing the image first

### OG image not showing
**Problem:** Edge runtime issue

**Solution:**
- Wait 30 seconds after creating the page
- Clear browser cache
- Check: `/api/og/congratulation?id=YOUR_ID`
